## 보완한 핵심 사항

초기 웹 서비스는 질환별 이진 모델의 확률을 비교해 상위 질환을 표시했다. 실제 테스트에서 궤양성 각막질환 이미지가 백내장으로 잘못 표시되는 문제가 있어, 최종 웹 추론은 정상 + 10개 질환 다중분류 모델을 우선 사용하도록 수정했다.

이 개선은 단순 구현을 넘어서 모델 출력 방식의 한계를 발견하고 해결한 과정이므로 보고서와 발표자료에서 기술적 난이도와 완성도를 설명하는 핵심 근거로 사용할 수 있다.
