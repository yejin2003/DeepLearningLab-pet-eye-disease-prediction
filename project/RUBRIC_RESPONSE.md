## 채점자 관점 핵심 메시지

이 프로젝트는 단순한 예측 스크립트가 아니라 데이터 선택, 전처리, 학습, 모델 한계 발견, 추론 방식 개선, 웹 서비스화, 제출 문서화까지 이어지는 완성형 딥러닝 실습 결과물입니다.

특히 질환별 이진 모델만 사용할 때 질환 간 순위가 흔들리는 문제를 실제 테스트 중 발견했고, 이를 정상 + 10개 질환 다중분류 모델로 보완했습니다. 이 과정은 프로젝트 수행 과정과 기술적 판단을 보여주는 중요한 보고서 포인트입니다.
